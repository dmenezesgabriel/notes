## Shortcuts

- **Search in editor**:

```sh
CTRL + F
```

- **Search and replace in editor**:

```sh
CTRL + H
```

- **Go to start of line**

```sh
Fn + LeftArrowKey
```

- **Go to end of line**

```sh
Fn + RightArrowKey
```

- **Go to line**:

```sh
CTRL + G
```

- **Split editors**:

```sh
CTRL + \
CTRL + P
# Choose a file to open on current editor
```

- **Navigate between split editors**:

```sh
CTRL + 1 # 2, 3, ...
```

- **Split a terminal**:

```sh
CTRL + SHIFT + 5
```

-**Navigate to right split terminal**:

```sh
ALT + RightArrowKey
```

-**Navigate to left split terminal**:

```sh
ALT + LeftArrowKey
```
