Domain Name Service

## Tipos de registros

- **A**: Mapeia um hostname para um IPv4
- **AAAA**: Mapeia um hostname para um IPv6
- **CNAME**: Mapeia um hostname para outro hostname (do tipo A ou AAAA)
- **NS**: Name Servers, sao os servidores que respondem as queries de DNS
