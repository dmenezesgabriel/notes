## O que é?

- Ambiente isolado
- Compartilha um host
- Todos os elementos necessários para rodar uma aplicação (código, runtime, bibliotecas, variáveis de ambiente, etc.)
- **LXC**: Linux Containers

## Vantagens

- Leve e portátil
- Facilita o ciclo de entrega

## Maquinas virtuais vs Containers

- Maquinas virtuais: cada máquina virtual tem seu próprio sistema operacional
- Containers: compartilham o mesmo sistema operacional, mas são isolados uns dos outros

## Tecnologias

- Docker
- Podman
