# Nginx

## Casos de uso

- Servidor Web
- Reverse Proxy (Porta de entrada)
- Api Gateway
- Load balancer

## Features

- Cache
- Filtro de requisições
- Controle de logs
- Controle de acessos

## Funcionamento

- Workers assíncronos invés de processos paralelos
