14:Tebf,<h2 id="o-que-é">O que é?</h2>
<p>Técnica para selecionar, modificar ou criar novos atributos a partir de dados existentes para melhor representar padrões. Isso inclui transformações matemáticas, combinações nos dados, extração de características temporais e etc.</p>
<p>Importante para lidar com ruídos, valores faltantes e tornar a informação relevante acessível aos modelos</p>
<ul>
<li>Seleção de atributos</li>
<li>Transformação de atributos</li>
<li>Criação de atributos</li>
<li>Codificação de variáveis categóricas</li>
<li>Normalização ou padronização</li>
</ul>
<h2 id="seleção-de-features">Seleção de Features</h2>
<ul>
<li>Escolher entre atributos existentes</li>
<li>Nem todos os atributos ou características dos dados são importantes para a modelagem</li>
</ul>
<h3 id="filtragem">Filtragem</h3>
<p>Uso de estatística para selecionar atributos baseando-se em seu relacionamento com a variável alvo</p>
<h3 id="wrapper-methods">Wrapper methods</h3>
<p>Uso de um modelo preditivo parta avaliar a importância do atributo</p>
<h3 id="embedded-methods">Embedded methods</h3>
<p>Integram a seleção dos atributos como parte do processo de treinamento do modelo. Ex: Modelos que utilizam regularização</p>
<h2 id="extração-e-criação-de-atributos">Extração e criação de atributos</h2>
<h3 id="análise-de-componentes-principais-pca">Análise de componentes principais (PCA)</h3>
<p>Reduz a dimensionalidade dos dados ao transforma-los em um conjunto menor de variáveis sumarias(components) que retém a maior parte da variabilidade original dos dados</p>
<h3 id="engenharia-de-atributos-temporais">Engenharia de atributos temporais</h3>
<p>Derivação de novos atributos a partir de variáveis de tempo como hora do dia, dia da semana, diferenciação de tempos entre eventos</p>

<li><a href="/data-science/machine-learning/target-encoding">target-encoding</a></li>

<h2 id="dados-faltantes">Dados Faltantes</h2>
<h3 id="imputação">Imputação</h3>
<p>Substituição de valores faltantes por um valor estimado com base em outras observações</p>
<h3 id="indicadores-de-dados-faltantes">Indicadores de dados faltantes</h3>
<p>Criação de novos atributos que indicam a presença de dados faltantes em outros atributos</p>
<h2 id="tratamento-de-dados-desbalanceados">Tratamento de dados desbalanceados</h2>
<h3 id="oversamplingundersampling">Oversampling/Undersampling</h3>
