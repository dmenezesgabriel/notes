14:Tdc9,<h2 id="bagging-bootstrap-aggregating">Bagging (Bootstrap Aggregating)</h2>
<p>Reduz a variância combinando modelos independentes como <em>Random Forest</em> <a href="/data-science/machine-learning/ensemble-method/random-forest">random-forest</a>, utilizado em detecção de fraudes.</p>
<p>Injeta aleatoriedade utilizando amostragem de inicialização (Bootstrap sampling) e fazendo a média dos resultados.</p>
<p>Utiliza estimativas de erro <em>out-of-bag</em></p>
<p><em>out-of-bag</em> objetos fora da amostra</p>
<h2 id="boosting">Boosting</h2>
<p>Reduz o viés corrigindo erros sequencialmente, bastante utilizado para predição de cliques e likes</p>
<h2 id="comparativo">Comparativo</h2>




























































<table><thead><tr><th align="left">Característica</th><th align="left"><strong>Bagging (Ensacamento)</strong></th><th align="left"><strong>Boosting (Impulsionamento)</strong></th></tr></thead><tbody><tr><td align="left"><strong>Objetivo</strong></td><td align="left">Reduzir a <strong>variância</strong> em modelos de alta variância.</td><td align="left">Reduzir o <strong>viés</strong> corrigindo sequencialmente os erros do modelo.</td></tr><tr><td align="left"><strong>Independência</strong></td><td align="left">Os modelos são treinados de forma <strong>independente</strong> e em paralelo.</td><td align="left">Os modelos são <strong>dependentes</strong>, com cada um corrigindo o erro do anterior.</td></tr><tr><td align="left"><strong>Ponderação</strong></td><td align="left">Todos os modelos têm <strong>peso igual</strong>.</td><td align="left">Os pesos são ajustados com base no <strong>desempenho</strong>.</td></tr><tr><td align="left"><strong>Dados de Treino</strong></td><td align="left">Cada modelo usa subconjuntos aleatórios com reposição (<strong>bootstrap</strong>).</td><td align="left">Cada modelo foca em pontos de dados classificados incorretamente pelo anterior.</td></tr><tr><td align="left"><strong>Exemplo Popular</strong></td><td align="left"><strong>Random Forest</strong>.</td><td align="left"><strong>AdaBoost</strong>, <strong>Gradient Boosting</strong>.</td></tr><tr><td align="left"><strong>Melhor Para</strong></td><td align="left">Modelos de alta variância, como árvores de decisão.</td><td align="left">Modelos de alto viés, onde ajustes sequenciais são úteis.</td></tr><tr><td align="left"><strong>Processo</strong></td><td align="left"><strong>Não iterativo</strong>; os modelos não dependem uns dos outros.</td><td align="left"><strong>Iterativo</strong>; cada modelo é treinado com base nos resultados anteriores.</td></tr><tr><td align="left"><strong>Combinação de Previsões</strong></td><td align="left">Agrega previsões por <strong>votação</strong> ou <strong>média</strong>.</td><td align="left">Combina previsões ponderadas com base na acurácia.</td></tr><tr><td align="left"><strong>Casos de Uso</strong></td><td align="left">Adequado para dados com mais ruído e variabilidade.</td><td align="left">Adequado para conjuntos de dados onde a melhoria da acurácia é necessária em várias iterações.</td></tr><tr><td align="left"><strong>Paralelismo</strong></td><td align="left">Processamento <strong>paralelo</strong> dos modelos.</td><td align="left">Processamento <strong>sequencial</strong> para correção de erros.</td></tr></tbody></table>
