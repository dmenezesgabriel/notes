JWT

Json Web Token

  • Token único
  • Nao modificável
  • Stateless
Authentication: Barer {JWT}

Um token jwt possui o seguinte formato: header.payload.sign

Header:

{
  "alg": "algoritmo",
  "typ": "JWT"
}

Payload:

{
  "sub": "subject (id do usuário)"
  // ...
}
