<h2 id="cidr">CIDR</h2>
<blockquote>
<p>Classless Inter-Domain Routing</p>
</blockquote>
<p>um método para alocar endereços de IP</p>
<p>composto por dois componentes:</p>
<ul>
<li>
<p><strong>Base IP</strong>:</p>
<ul>
<li>Representa um IP contido no intervalo (xx.xx.xx.xx) Ex: 10.0.0.0, 192.168.0.1</li>
</ul>
</li>
<li>
<p><strong>Subnet Mask</strong>:</p>
<ul>
<li>Define quantos bits podem mudar no IP. Ex: /8, /16, /24, /32, ...</li>
</ul>
</li>
</ul>
