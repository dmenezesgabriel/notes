14:T7a0,<h2 id="casos-de-uso">Casos de Uso</h2>
<ul>
<li>Alternativa ao Apache Kafka</li>
<li>Mover dados rapidamente de produtores de dados e, em seguida, processá-los continuamente, seja para transformá-los antes de emiti-los para um armazenamento de dados,</li>
</ul>
<h2 id="features">Features</h2>
<ul>
<li>Dados em <em>tempo real</em> (Stream)</li>
<li>Retenção de dados por 365 dias</li>
<li>Dados não podem ser deletados até que expirem</li>
<li>Dados até 1MB</li>
<li>Criptografia KMS/HTTP</li>
<li>Kinesis Producer Library (KPL) para escrever uma aplicação producer otimizada</li>
<li>Kinesis Client Library (KCL) para escrever uma aplicação consumer otimizada</li>
</ul>
<h2 id="arquitetura">Arquitetura</h2>
<garden-mermaid diagram="architecture-beta

<h2 id="provisioned-mode">Provisioned Mode</h2>
<ul>
<li>Escolha do numero de <code>shards</code></li>
<li>Cada <code>shard</code> recebe até 1MB/s (ou 1000 registros por segundo)</li>
<li>Cada <code>shard</code> pode enviar até 2MB/s</li>
<li>Escala manual do numero de <code>shards</code></li>
<li>Pagamento por <code>shards</code> provisionada por hora</li>
</ul>
<h2 id="on-demand-mode">On-demand mode</h2>
<ul>
<li>Não é necessário provisionar ou gerenciar a capacidade</li>
<li>Capacidade padrão provisionada de entrada (4MB/s ou 4000 registros por segundo)</li>
</ul>
