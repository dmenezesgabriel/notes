Key Management Service

  • O escopo de chaves KMS é regional, caso necessário copiar um snapshot de EBS ebs, por exemplo, é necessário realizar a criptografia com uma chave diferente.

Features

  • Gerenciado pela AWS
  • Integrado com IAM iam para autorizações
  • Maneira simples de controlar acesso aos dados
  • Possível auditar chaves KMS pelo Cloud Trail cloud-trail
  • Integração com grande parte dos serviços da AWS (EBS, S3, RDS, SSM)
  • SDK e CLI
